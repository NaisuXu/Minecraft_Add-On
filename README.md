# Minecraft_Add-On

A useful repository of add-on packages.

## behavior_packs

 - better_wandering_trader
    Changed the wandering trader trade list to now buy rare items such as diamonds, ancient_debris, villager_spawn_egg, nether_star, heart_of_the_sea, elytra, etc

 - creeper_no_break_blocks
    Creepers no longer break blocks.
